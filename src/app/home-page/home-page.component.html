Weather Forecast Page using OpenWeatherMap API

On this site you can get a 5 day or 16 day weather forecast

5 day weather forecast

5 day forecast is available at any location or city. It includes weather data every 3 hours. Also you can choose some filters:

  • Humidity, %
  • Pressure, hPa
  • Current temperature, ℃
  • Maximum daily temperature, ℃
  • Minimum daily temperature, ℃
  • Wind direction
  • Wind speed, m/s

16 day weather forecast

16 day forecasts is available at any location or city. Forecasts include only daily weather. Also you can choose some filters:

  • Humidity, %
  • Pressure, hPa
  • Day temperature, ℃
  • Night temperature, ℃
  • Maximum daily temperature, ℃
  • Minimum daily temperature, ℃
  • Morning temperature, ℃
  • Evening temperature, ℃
  • Wind direction
  • Wind speed, m/s
